# NextFut Backend

Backend da plataforma NextFut.

## Stack

- Node.js
- Express
- Prisma
- PostgreSQL

## Responsabilidades

- API REST
- Cadastro de atletas
- Cadastro de avaliações
- Cálculo de score
- Recomendações de treino
- Importação CSV

## Desenvolvimento

Instalar dependências:

npm install

Rodar em desenvolvimento:

npm run dev

Acessar healthcheck:

http://localhost:3000/api/health
